IPv4 Multicast MAC Addresses

- MAC address is 48-bit

- The most significant 24 bits of an IPv4 multicast MAC address are 0x01005E

- 25th-bit of IPv4 multicast MAC-Address is ZERO

- IPv4 multicast range is 224.0.0.0/4 so 1110 are always fixed

IPv6 Multicast MAC Addresses

- MAC address is 48-bit

- The most significant 16 bits of an IPv6 multicast MAC address are 0x3333

- IPv6 multicast range is FF::/8 so 8 bits are always fixed

IPv6 NAT Types

  • Stateless NAT
    • Translation Mechanism Algorithmicallly
    • Not keeping Any Binding Or Any Session State
    • 1:1 Translation
    • Types:
      • NPTv6

  • Stateful NAT
    • Keeping The Session State Or Binding
    • 1:N Translation
    • Types:
      • Static NAT64
      • Dynamic NAT64
  • NAT64
    • To Communicate IPv6 Host With IPv4 Host
    • Need To translate Source and Destination Address
    • Prefix 64:FF9B::/96 is allocated for NAT64 by IANA
  • NAT66 => NPTv6

DNS64

DNS64 is a DNS service that returns AAAA records with synthetic IPv6 addresses for IPv4-only destinations.

DNS64 actions:

  • Receives AAAA type query
  • Tries to get Ipv6 address
  • When the answer is empty
  • DNS64 tries to get IPv4 address
  • Creates the IPv6 address by Nat64-Prefix and IPv4 address
  • Responds to the client the synthetic-IPv6 address
