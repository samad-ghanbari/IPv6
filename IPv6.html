IPv6 Practical Applications

  • ping
    • ping ::1
    • ping -6 ::1
  • Network Share
    • in ipv4: \\10.82.14.38\share
    • in ipv6: \\2001-1234-1234--1\share
  • Web Page
    • [::1]
    • http://[::1]
    • https://[2000::1]
    • http://[2001::1]:8080

IPv6 Format

X:X:X:X:X:X:X:X / Y

X is a 16-bit hexadecimal

no subnet-mask

We Use Prefix-Length

2200:1234:5678:ABCD:1234:5678:ABCD:1234/64

2200:1234:5678:ABCD:: Prefix
1234:5678:ABCD:1234 Host
64 Prefix-Length

Bitwise Group

1 bit Bit
2 bit Duet
3 bit Trio
4 bit Nibble/Quartet
5 bit Quintet
6 bit Sextet
7 bit Septet
8 bit Byte/Octet/Character
9 bit Nonet
10 bit Declet
16 bit Hexadectet/Hextet/Doublet
32 bit Quadlet

128 bit Hexlet
