IPv6 Format

X:X:X:X:X:X:X:X / Y

X is a 16-bit hexadecimal

no subnet-mask

We Use Prefix-Length

2200:1234:5678:ABCD:1234:5678:ABCD:1234/64


2200:1234:5678:ABCD:: Prefix
1234:5678:ABCD:1234 Host

Find the 43th bit Hextet 2*16=32 && 3*16=48 => 2*Hextet + 2*nibble + 3bit

IPv6 Address Types

  • Unicast
    • GUA - Global Unicast - 2000::/3
    • ULA - Unique Local - FC00::/7 ~ FD00::/8
    • LLA - Link-Local - FE80::/10
    • Loopback - ::1/128
    • Unspecified - ::
  • Multicast - FF::/8
  • Anycast
